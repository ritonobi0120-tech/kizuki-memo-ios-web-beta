# Kizuki Memo iPhone Web Beta

Windows だけで配布できる、iPhone 向けの Web beta です。

## 使い方

1. iPhone の Safari で公開 URL を開く
2. 共有メニューから `ホーム画面に追加`
3. ホーム画面の `きづきメモ beta` を開く
4. 名前を選んで、メモ欄でキーボードのマイクを使って話す

## 注意

- これはネイティブ iPhone アプリではなく、ホーム画面追加で使う Web beta です
- データはその iPhone のブラウザ保存領域に入ります
- 念のため `設定 > JSON を書き出す` で控えを残せます
